# K.K book — Premium Social Networking Service

ガクチカ用に作成された、X（旧Twitter）やFacebookのようなモダンで洗練されたUX/UIを誇る、プレミアムなSNSウェブアプリケーションです。
外部CSS（Tailwindなどのフレームワークを使わないバニラCSS）で構築された美しいデザインシステム、滑らかなSVGインタラクション、そして一瞬のちらつきもない完璧なダーク/ライトテーマ切り替え機能を搭載しています。

---

## ✨ 主な機能と特徴

### 1. 🎨 プレミアム・マルチテーマシステム（ダーク＆ライト）
* **心地よいビジュアル**: 高級感のあるスレートグレーベースの「ダークモード」と、目に優しく洗練されたソフトグレーベースの「ライトモード（白基調）」の2つのテーマを搭載。
* **テーマ自動保存**: 切り替えたテーマは `localStorage` に保存され、次回アクセス時も維持されます。
* **ちらつき防止（Anti-Flash）設計**: HTMLの `<head>` 内で初期テーマをロードするIIFE（即時実行関数）を実行することで、ライトモード選択時のページ読み込みの瞬間にダークモードが一瞬見えてしまう「画面のフラッシュ現象」を完全に排除しています。
* **テーマ間のスムーズトランジション**: ボタン、カード、背景、入力枠などすべての要素が約0.3秒で優しく溶けるようにフェード遷移します。

### 2. 📱 モダンな2カラムレイアウト ＆ レスポンシブ設計
* **スマートなグリッド構成**: 
  * **左ナビゲーションサイドバー**: アプリロゴ、ホーム・通知・メッセージ等のナビメニュー、サイドバー統合テーマスイッチ、下部にはログイン中ユーザーのプロフィールとログアウトボタン。
  * **中央メインタイムライン**: 十分の余白と美しいカードで構成された投稿フィード、および直感的なインライン返信エリア。
* **マルチデバイス対応**: スマートフォン、タブレット、デスクトップなどあらゆる画面サイズで美しくレイアウトが最適化されます。スマートフォン表示ではサイドバーが折りたたまれ、フィードが全幅に広がります。

### 3. ⚡ 絵文字を排除した、高品質なSVGアクション ＆ マイクロアニメーション
* **「いいね（Like）」機能**: カスタムSVGハートを採用。いいねを押すと、リアルな鼓動のようにハートが跳ねる（パルスアニメーション）とともに深みのあるクリムゾンレッドに染まります。
* **「返信（Reply）」機能**: 吹き出しSVGアイコンを採用。クリックすると、該当ポストのインライン返信フォームに滑らかにフォーカスが当たります。
* **インラインスレッド表示**: 返信は投稿カードの直下にネストして、投稿日時の新しい順（降順）でタイムラインを崩さずに綺麗に整列します。

### 4. 🔒 安全な認証 ＆ グラスモルフィズム
* **美しいログイン/新規登録画面**: 曇りガラスのように背景を美しくぼかすグラスモルフィズムカードを導入。
* **セキュリティ**: `werkzeug.security` によるパスワードのセキュアなハッシュ化（PBKDF2-SHA256）と、`Flask-Login` を用いた堅牢なセッション管理。

---

## 🛠️ 技術スタック

* **サーバーサイド**: Python 3 / Flask
* **データベース**: SQLite / Flask-SQLAlchemy (ORM)
* **認証**: Flask-Login
* **フロントエンド**: HTML5 / CSS3（CSSカスタムプロパティ・デザインシステム変数） / Vanilla JavaScript（ゼロ依存）

## 🚀 セットアップ ＆ 起動手順

### 1. 仮想環境の作成とライブラリのインストール
```bash
# 仮想環境を作成
python3 -m venv .venv

# 仮想環境をアクティベート
source .venv/bin/activate  # macOS/Linux の場合
# .venv\Scripts\activate   # Windows の場合

# 依存パッケージをインストール
pip install -r requirements.txt
```

### 2. データベースの初期化とサーバーの起動
```bash
# アプリケーションを起動（初回起動時にSQLiteデータベース `sns.db` が自動生成されます）
python app.py
```

### 3. ブラウザでアクセス
ターミナルに表示されるローカルURL（通常は [http://127.0.0.1:5000](http://127.0.0.1:5000)）にアクセスします。まずは「新規登録」からアカウントを作成して、プレミアムなSNS体験をお楽しみください！
